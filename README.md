## 🚀 Comece Aqui!

### 📌 Como Usar

- 🔍 **Navegue** pelos diretórios por tópico  
- 💻 **Escolha** exercícios pela dificuldade (⭐️ = Fácil, ⭐️⭐️ = Médio, ⭐️⭐️⭐️ = Desafiador)  
- 🧠 **Pratique** resolvendo os desafios  
- 🔄 **Compare** suas soluções com as respostas sugeridas
